## Installation

```sh

pipenv install --dev

```


## How to Run

Create MkDocs project

```sh

mkdocs new . 

mkdocs serve

```

Install MkDocs Bootstrap

```sh

pipenv install -d mkdocs-bootswatch

```

Install MkDocs Material

```sh

pipenv install -d mkdocs-material

```

Make MkDocs build

```sh

mkdocs build

```

Make deploid on github

```sh

mkdocs gh-deploy

```
